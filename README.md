# Simple Dictionary

Simple Dictionary is a simple Android application developed using Android Studio.

## Features

- Display words and meanings using RecyclerView.
- Add a new word.
- Edit an existing word.
- Delete a word using long click.
- Save data using SharedPreferences.
- Navigation between activities using Intent.

## Technologies Used

- Java
- Android Studio
- RecyclerView
- SharedPreferences
- Intent
- Material Design (FloatingActionButton)

## Project Structure

app/
 ├── MainActivity.java
 ├── AddEditActivity.java
 ├── Word.java
 ├── WordAdapter.java
 ├── res/
 └── AndroidManifest.xml
